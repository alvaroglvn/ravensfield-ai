<task>

1. Be written in conservation documentation voice (clinical, present-tense, observational)
2. Be optimized for FLUX image generation (visual, spatial, geometric descriptions)

<conservation_voice>
**Voice Characteristics:**

ALWAYS use:
- Present-tense observational verbs: "exhibits," "shows," "displays," "presents," "features," "reveals"
- Technical material and condition terminology
- Spatial and geometric descriptions
- Precise visual observations
- Clinical, unemotional tone
- Official documentation framing

NEVER use:
- Past-tense narrative ("was created," "belonged to")
- Emotional language ("haunting," "mysterious," "eerie," "ominous")
- Metaphors or similes ("like," "resembling," "as if")
- Interpretive language ("suggests," "evokes," "implies")
- Abstract concepts ("ancient," "spiritual," "otherworldly")
</conservation_voice>

<flux_optimization>
**What FLUX Understands:**

✅ FLUX CAN interpret:
- Relative scale: "palm-sized," "human-scale," "miniature," "life-sized," "monumental"
- Visual properties: color, texture, finish, transparency
- Spatial relationships: "extending from top to bottom," "centered," "arranged in triangular pattern"
- Geometric descriptors: "vertical," "circular," "asymmetric," "radiating"
- Prominence indicators: "prominent," "subtle," "deep," "shallow," "extensive"
- Material surface states: "granular," "smooth," "pitted," "glossy," "matte"
- Lighting patterns: "diffused," "even," "softbox," "ring flash"
- Composition cues: "full-frame," "macro close-up," "centered with negative space"

❌ FLUX CANNOT interpret:
- Absolute measurements (8cm, 25mm) - include for authenticity but pair with visual descriptors
- Precise quantities beyond basic counting (three = yes, seventeen = no)
- Emotional states ("sad," "angry," "peaceful")
- Abstract concepts ("ancient," "mystical," "cursed")
- Historical context (dates, provenance, ownership)
- Metaphorical language
</flux_optimization>

**2. MATERIAL IDENTIFICATION (1-2 sentences):**

Pattern: [Material type] with [visual surface properties] [forming/depicting] [object type]

CRITICAL: Use "forming" or "depicting" NOT "is":

<uncanny_elements>
**Creating Weirdness Through Visual Specificity:**

The artifact should have one element that's subtly impossible or wrong. This must be VISUALLY OBSERVABLE and described clinically.

Types of visual impossibilities:

1. **Wrong Quantity:** Incorrect count (three eyes, six fingers, no mouth opening)
2. **Absent Element:** Missing expected feature (smooth void where face should be)
3. **Geometric Impossibility:** Pattern that defies logic (perfectly circular crack, symmetrical organic form)
4. **Scale Violation:** Wrong proportions (one eye larger than other, miniature monumental detail)
5. **Directional Wrong:** Oriented impossibly (shadow wrong direction, grain flowing backward)

**In conservation voice, describe the impossibility CLINICALLY without editorializing:**

- ✅ "exhibits three circular ocular cavities arranged in triangular configuration where bilateral symmetry would be expected"
- ❌ "has a creepy third eye that shouldn't exist"

The weirdness emerges from precise clinical description of something that's subtly wrong.
</uncanny_elements>
